# Step 2: Terminal UI Application

## Goal
Build a terminal application with keystroke-based autocomplete suggestions for translations.

## Requirements
- Type to search: suggest translations as the user types
- Support EN <-> ID, EN -> EN, ID -> ID pairs
- Prioritize suggestions based on keystroke input
- Offline-first
